# Habit Heatmap 🔥

A GitHub-style contribution heatmap built using pure HTML, CSS, and JavaScript.

This project dynamically generates 365 squares representing daily activity and visualizes intensity using color levels.

---

## 🚀 Features

- Generates 365 days dynamically using JavaScript
- Random activity simulation (0–8 activities per day)
- Color intensity levels (like GitHub contributions)
- Hover tooltip showing activity count
- Responsive CSS Grid layout
- Clean and minimal UI

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Grid, positioning)
- JavaScript (DOM manipulation, loops, event listeners)

---

## 🧠 What I Learned

- Creating elements dynamically with JavaScript
- Using loops to generate UI components
- Conditional class assignment for styling
- Handling mouse events (mouseenter, mousemove, mouseleave)
- Building tooltip behavior from scratch
- Debugging small but critical typos in JavaScript
